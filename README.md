# Tic Tac Toe Game

A classic Tic Tac Toe game built with HTML, CSS, and JavaScript. Enjoy this timeless two-player game with a clean and responsive interface.

## 🎮 Live Demo

[Play the game!](#) https://tic-tac-toe-omega-sooty.vercel.app/

## ✨ Features

- **Two-player gameplay** - Take turns between X and O
- **Clean, responsive design** - Works on desktop, tablet, and mobile devices
- **Visual feedback** - Clear winner announcement and draw detection
- **Game controls** - Reset button and new game option
- **Win detection** - Automatically detects all winning combinations
- **Draw detection** - Identifies when the game ends in a draw
- **Smooth animations** - Interactive hover effects and transitions

## 🎯 How to Play

1. Players take turns clicking on the empty boxes
2. First player uses **O**, second player uses **X**
3. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins
4. If all nine boxes are filled without a winner, the game ends in a draw
5. Click **"Reset Game"** or **"New Game"** to start over

## 🛠️ Technologies Used

- **HTML5** - Structure and layout
- **CSS3** - Styling, animations, and responsive design
- **JavaScript** - Game logic, win detection, and interactivity
